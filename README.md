# ReCollect

## An inventory app for your household essentials

### Setup

* Clone Repo
* Run `npm install`
* Run `bundle`

#### Create Database

* `rails db:drop`
* `rails db:create`
* `rails db:migrate`
* `rails db:seed`

### Technologies

* Ruby Version: 2.6.1

* Rails

* React Router
